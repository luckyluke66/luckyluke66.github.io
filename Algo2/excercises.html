Úkol 1 – Adresář skladeb

Následující úkol pokrývá základní znalosti, které byste už měli mít z kurzu Jazyk C a Algoritmy 1.

Struktura struct song

Vytvořte v jazyce C strukturu struct song pro záznam skladeb obsahující:

  • název skladby (title)
  • interpreta (artist)
  • délku skladby v sekundách (length)
  • rok vydání (year)
  • hodnocení skladby (1–10) (rating)

Databáze skladeb

Vytvořte databázi skladeb jako pole struktur song (data si vymyslete, alespoň pět záznamů).

Přístupové funkce

Napište následující funkce (parametr n je vždy velikost pole):

string song_title(struct song* s);
string song_artist(struct song* s);
int song_length(struct song* s);
int song_year(struct song* s);
int song_rating(struct song* s);

Pokud ještě neznáte pointery, můžete tyto funkce vynechat a přistupovat k položkám přímo přes songs[i].title apod.

Práce s databází skladeb

Napište následující funkce:

Výpis délky skladby

Vypíše délku skladby na standardní výstup pro zadaný název a interpreta:

void print_length(struct song songs[], int n, string title, string artist);

Nejlépe hodnocená skladba

Vrátí index skladby s nejvyšším hodnocením:

int best_rated(struct song songs[], int n);

Zvýšení hodnocení skladby

Zvýší hodnocení zadané skladby o danou hodnotu (max 10):

void increase_rating(struct song songs[], int n, string title, string artist, int amount);

Snížení hodnocení skladby

Sníží hodnocení zadané skladby o danou hodnotu (min 1):

void decrease_rating(struct song songs[], int n, string title, string artist, int amount);

i-tá nejkratší skladba

Vrátí index i-té nejkratší skladby:

int ith_shortest(struct song songs[], int i, int n);

Třídění podle roku vydání

Vypíše databázi skladeb setříděnou vzestupně podle roku vydání:

void sort_songs(struct song songs[], int n);
